====================================================================
 O QUE FOI INSTALADO
====================================================================

  KOM 2.0 e um protocolo, nao um software. Nada foi "instalado"
  no sentido tradicional. Os arquivos abaixo foram copiados para
  seu projeto:

  ARQUIVOS PRINCIPAIS:
  - AGENTS.md        -> Instrucoes para o agente de IA seguir
  - README.md        -> Documentacao completa do protocolo
  - kom/             -> 9 documentos com as 6 fases do ciclo
  - .opencode/skills/ -> 6 skills para auto-ativacao no OpenCode

  MEMORIA DO PROJETO (criada pelo agente):
  - knowledge/registry/  -> Decisoes arquiteturais (ADRs)
  - knowledge/lessons/   -> Licoes aprendidas
  - knowledge/patterns/  -> Padroes identificados

====================================================================
 COMO USAR
====================================================================

  1. Abra seu terminal com OpenCode (ou Claude Code, Codex, etc.)
  2. O agente de IA vai ler o AGENTS.md automaticamente
  3. Diga o que voce quer fazer, ex:

     "crie uma API de autenticacao"
     "refatore o modulo de pagamentos"
     "precisa de ajuda com a arquitetura do projeto"

  4. O agente seguira as 6 fases do KOM automaticamente

  FASES DO CICLO:
    1. Orientacao  -> Entender o problema
    2. Arquitetura -> Decidir a estrutura
    3. Contrato    -> Especificar interfaces
    4. Execucao    -> Implementar
    5. Verificacao -> Auditar a entrega
    6. Registro    -> Salvar decisoes e aprendizados

====================================================================
 GRAPHIFY (GRAFO DE CONHECIMENTO - OPCIONAL)
====================================================================

  O Graphify cria um grafo do seu codigo para consultas rapidas.
  Ele NAO vem instalado. Para instalar:

  >>> pip install graphifyy

  (sem API key, funciona em modo AST-only, sem custo)

  Se quiser, peca para o agente de IA:

     "instale o graphify para mim"
     "configure o grafo de conhecimento"

  Ele sabe exatamente o que fazer.
