# KOM 2.0 — Registry

## Consultar

Antes de decidir, consulte `knowledge/registry/` para decisões anteriores relacionadas.

## Registrar

Depois de decidir, crie um arquivo em `knowledge/registry/` com:

```
Contexto:  por que foi necessária
Decisão:   o que foi decidido
Alternativas: o que foi descartado
Motivo:    por que esta opção venceu
Impacto:   o que muda
```
